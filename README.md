# 魅魔的网站 Workspace

## 目录总览

- website_nodejs_meimo/: 主站点项目（Node.js + Express）
- docs/: 根目录级文档与需求资料
- archive/: 历史页面、备份与实验样例
- resources/: 设计素材与品牌资源

## 常用命令（在根目录执行）

```bash
npm run dev
npm run start
```

## 说明

- 页面样式与线上路径保持不变。
- 运行时代码集中在 website_nodejs_meimo/server.js、website_nodejs_meimo/src/、website_nodejs_meimo/public/。
- archive/ 与 resources/ 中的文件默认不参与运行，仅做归档与参考。
